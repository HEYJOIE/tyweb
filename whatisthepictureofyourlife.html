What's the picture of our life?

by taiyang Ocv 18th, 2016

Time is a real phenomenon, a continuous change through which we live. We can perceive the changing time by sunrise, sunsets, night and day, the changing seasons, and we can also record them by taking photos or videos. But have you ever considered such a question, "What's the picture of your life?"

We use sensors and Arduino to collect data. Then, we imported the data into Excel and calculated the average value of it for every 6 records (1 minute) by using Python, then generated new records. Using RGB mode to represent each record. RGB mode is a scale from 0 to 255 and three values can represent a kind of color. In our case, we also used time, light and temperature to determine a color. We set the time of the ﬁrst record as 0, and the original data of each record could be a list like (time, light, temperature), we used division and got a new list like (time%256, light%256, temperature%256). In the calculation formula of “Data% 256”, ”%” can stand for a modulo symbols. So, every 60 new records can generate a long rectangular bar which can represent an hour, and the 24 bars can be formed into the following picture to present the whole day.
